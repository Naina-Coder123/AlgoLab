# Sorting Visualizer

## 📌 Overview

The **Sorting Visualizer** is a web-based application that visually demonstrates how different sorting algorithms work. It provides an interactive way to understand the internal working of sorting algorithms by displaying animated steps showing how elements are compared, swapped, and arranged.

The project is designed as an educational tool to help students and developers better understand algorithm behavior and performance through visualization.

---

## 🚀 Features

* Visual representation of sorting algorithms
* Real-time animation of sorting steps
* Generate random arrays for visualization
* Adjustable sorting speed for better understanding
* Interactive and user-friendly interface
* Helps visualize algorithm efficiency and behavior

---

## 🧠 Algorithms Implemented

The application currently supports visualization of the following algorithms:

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

Each algorithm is animated step-by-step to demonstrate how the sorting process works.

## ⚙️ Installation and Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/sorting-visualizer.git
```

### 2. Navigate to the project directory

```bash
cd sorting-visualizer
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
